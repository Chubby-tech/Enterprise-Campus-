# Third Networking Project – Enterprise Campus Design

## 📌 Overview
This project is a Cisco Packet Tracer enterprise campus network design for a multi-floor organization. It demonstrates VLAN segmentation, redundant routing, DHCP services, and full wireless deployment across all departments.

---

## 🏢 Network Structure

### Floor 1
- Reception → 192.168.8.0/24  
- Store → 192.168.7.0/24  
- Logistics → 192.168.6.0/24  

### Floor 2
- Finance → 192.168.5.0/24  
- HR → 192.168.4.0/24  
- Sales → 192.168.3.0/24  

### Floor 3
- Admin → 192.168.2.0/24  
- IT Department → 192.168.1.0/24  

---

## 🔀 Routing Design
- 3 Router Redundant Topology (RF1, RF2, RF3)
- Serial Links:
  - 10.10.10.0/30
  - 10.10.10.4/30
  - 10.10.10.8/30
